HarvardX: PH125.9x: Data Science - Capstone - Choose Your Own Project - Adult Census Income Project

This repository is created to file and share one of the two capstone projects of the HarvardX Professional Certificate of Data Science.

This project is to predict whether income exceeds $50k/year using census data in the Adult Census Income dataset (Dateset source: https://www.kaggle.com/datasets/uciml/adult-census-income).
